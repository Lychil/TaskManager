# 🚀 Task Manager - Тестовое задание

## 📝 Описание

Приложение для управления задачами, разработанное в рамках тестового задания. Реализованы основные CRUD-операции с задачами, фильтрация и сортировка.

## 🛠 Технологический стек

### Клиентская часть
- **React 18** (функциональные компоненты, хуки)
- **TypeScript** для типизации
- **Vite** как сборщик
- **Styled Components** для стилизации
- **Axios** для HTTP-запросов
- **React Router** для навигации
- **Vitest** для unit-тестов

### Серверная часть
- **Node.js** + **Express**
- **json-server** как mock API

## 🖥 Запуск
```powershell
# Запуск сервера (из папки server)
npm run start

# Запуск клиента (из папки client)
npm run dev

# Запуск тестов (из папки client)
npm run test
```
